# Une_application_bancaire_par_Tkinter
# 🏦 Système de Gestion Bancaire Interactif

Une application de bureau développée en **Python** avec **Tkinter**, mettant en œuvre les concepts avancés de la **Programmation Orientée Objet (POO)**.

## Fonctionnalités:
- **Modes de Paiement multiples** : Carte Bancaire (1.5% frais), PayPal (2% frais), et Crypto Wallet (0.5% frais).
- **Validation Intégrée** : Vérification des formats d'emails, numéros de cartes (16 chiffres) et adresses Crypto.
- **Tableau de Bord Statistiques** : Suivi en temps réel du solde, des dépôts, des retraits et des frais cumulés.
- **Historique Dynamique** : Journalisation complète des transactions avec horodatage.
- **Interface Dark Mode** : Design moderne optimisé pour la lisibilité.

## Architecture POO:
Le projet démontre l'utilisation de :
- **Classes Abstraites** (`ABC`) pour définir les contrats de paiement.
- **Encapsulation** pour la protection des montants (Getters/Setters).
- **Héritage & Polymorphisme** pour spécialiser les méthodes de calcul des frais.
